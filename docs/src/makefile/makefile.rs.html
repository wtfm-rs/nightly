3//! # Makefile macros
4//!
5//! Since we use make as the top level CLI/CI, it would be convenient to 
6//! wrap Rust nigthly toolchain commands with 
7//! [Makefile macros](https://www.gnu.org/software/make/manual/html_node/Using-Variables.html).
8//! 
9//! ```make
10//! RUSTC = rustup run nightly rustc
11//! RUSTDOC = rustup run nightly rustdoc --out-dir docs
12//! CARGO = rustup run nightly cargo
13//! ```
14//! Here is how to use it.
15//!
16//! `make versions`
17//! ```make
18//!	$(RUSTC) --version --verbose
19//!	$(RUSTDOC) --version --verbose
20//!	$(CARGO) --version --verbose
21//! ```
22//! Local MacOS (You can see CI output in [workflow](../workflow/index.html).)
23//! ```sh
24//! % make versions
25//! rustup run nightly rustc --version --verbose
26//! rustc 1.95.0-nightly (eda76d9d1 2026-01-21)
27//! binary: rustc
28//! commit-hash: eda76d9d1d133effbf7facb28168fd78d75fd434
29//! commit-date: 2026-01-21
30//! host: aarch64-apple-darwin
31//! release: 1.95.0-nightly
32//! LLVM version: 21.1.8
33//! rustup run nightly rustdoc --out-dir docs  --version --verbose
34//! rustdoc 1.95.0-nightly (eda76d9d1 2026-01-21)
